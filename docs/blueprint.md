# **App Name**: CalorieWise

## Core Features:

- Food Input: Allow users to input the name of the food and quantity they consumed, specifying units such as weight or cups.
- Calorie Calculation: Calculate the calorie count of the entered food based on the provided quantity using a nutritional database.
- Daily Breakdown: Display a breakdown of calorie intake for each meal (breakfast, lunch, dinner, snacks) throughout the day.
- Over Consumption Warning: Alert the user if their calorie intake exceeds the recommended daily allowance, with an option for a AI-powered meal suggestion for the next day.
- AI Meal Planner: Based on user's preferred foods, restrictions, and daily calorie goals, the app uses a tool to generate a tailored meal plan for the next day, giving precise options to follow.
- Calorie History: Store the daily calorie intake details in a Firestore database, allowing users to track their dietary history.
- Monthly Analysis: Calculate the average daily calorie intake for the month and display a timeline dashboard visualizing calorie trends over the month.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) to evoke a sense of health and well-being.
- Background color: Very light green (#F0F4EF), subtly desaturated.
- Accent color: Muted yellow (#D4AC0D), approximately 30 degrees to the 'left' of the primary color on the color wheel, for a natural, complementary highlight.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for headlines or body text.
- Use clear and simple icons to represent food groups and activities.
- Design a clean, intuitive dashboard layout that prioritizes calorie information and visual data representation.
- Use subtle transitions and animations to enhance the user experience when navigating and inputting data.
